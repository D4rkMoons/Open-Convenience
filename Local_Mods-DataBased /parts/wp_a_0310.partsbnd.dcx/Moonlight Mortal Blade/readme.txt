Instructions to install
1. If you don't have Mod Engine installed, please do so by visiting this link httpswww.nexusmods.comsekiromods6.

2. Make a folder called 'mods' in your Sekiro directory (where your .exe is) if you haven't already.

3. In the mods folder, make a new folder called 'parts.'

4. Place the files in the parts folder and you are good to go!
