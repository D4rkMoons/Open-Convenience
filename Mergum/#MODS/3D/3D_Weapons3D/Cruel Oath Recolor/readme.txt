INSTALL:

Mod Engine 2
1. Install ModEngine2 ( https://github.com/soulsmods/ModEngine2/releases )
2. Put file .partsbnd inside the mod\parts folder
3. Run the game

Metis Mod launcher
1. Install ModEngine2 ( https://www.nexusmods.com/eldenring/mods/3102 )
2. Put file .partsbnd inside 'FOLDER'\parts
3. Launch Metis
4. Create new mod
5. Create new profile 
6. Select your mod in profile
3. Run the game.
